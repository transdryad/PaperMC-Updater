Welcome to PaperMC-Updater's documentation!
===================================

**PaperMC-Updater** is a Python script for managing a minecraft server
that has an inbuilt package manager for plugins.

Check out the :doc:`usage` section for further information, including
how to :ref:`install` the project.

.. note::

  **This project is under active development! Features may chage or be removed at any time! Not for production usage!**
