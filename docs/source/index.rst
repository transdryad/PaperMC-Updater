Welcome to PaperMC-Updater's documentation!
===================================
.. note::

    **These docs are under construction.**

**PaperMC-Updater** is a Python script for managing a minecraft server
that has an inbuilt package manager for plugins.

Check out the :doc:`usage` section for further information, including :ref:`install` for the project.

.. warning::

  **This project is under active development! Features may change or be removed at any time! Not for production usage!**
Thanks to PaperMC for their minecraft server software.
